## Merge Modules
- Able to merge two identical module to reduce duplicate ones.
- `Target` one is the expected one after merge, `source` one is the one about to be used to merge.
- Merge modules function can locate in Modules section at the top menu bar of the Tosca Commander tool.

## Value Range
- Separate by `;` sign, no spaces required.
- Help varsities values able to be used in the `value` column.
- Used by select from the dropdown of the `value` cell.
